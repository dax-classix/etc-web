# Ethereum Classic Web

## Usage

### Develop

1. Clone this repo
2. `npm install`
3. `npm start`

### Deploy

**Manual Push:**

1. `npm run build-prefix-links`
2. Copy contents of `/public`
3. Checkout `gh-pages`
4. Paste & replace contents
5. Push

**Automated Checkout:**

Install [`gh-pages`](https://www.npmjs.com/package/gh-pages)

1. `npm run deploy`
